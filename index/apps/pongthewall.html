Pong The Wall (Processing and P5.js)

2021

In 2021, I took a class in Creative Coding that taught me how to use Processing (Java). I combined elements from the pupular arcade games "Break Out" and "Pong!" to write 'Pong The Wall!'.

Documentation




I put this game together to learn object oriented programming. My biggest challenge while debugging is hitting the ball with the paddle while both are moving toward each other. I am proud to have created a game that matches my initial conception.
